# parkingBot

Telegram bot koji prikazuje trenutan broj slobodnih parking mesta na 26 parkiralista/garaza u Beogradu. Moguc prikaz broja slobodnih mesta na svim dostupnim parkiralistima i pretraga najblizeg parkiralista sa slobodnim mestima od zadate lokacije.
Kao metrika "blizine" koristi se duzina voznje od zadatke tacke do parkiralista, dobijena preko Googlovog distance matrix APIja.
